**NIP-36 Content-Warning Solution for Age-Gating**

Spec: `["content-warning", "optional reason text"]` tag in Nostr events

**Implementation Plan**:
1. Add `contentWarning: String?` field to VideoEvent model
2. Parse `content-warning` tag in VideoEvent.fromNostrEvent() around line 100-300
3. Filter in videoEventsProvider and homeFeedProvider:
   - If video has contentWarning → check AgeVerificationService.isAdultContentVerified

4. Store warning reason for potential UI display

**Tag Parsing Location**: In video_event.dart switch statement around line 99-300 where we parse tags like 'title', 'duration', etc.

**Filter Location**: 
- lib/providers/video_events_providers.dart (Explore feed)
- lib/providers/home_feed_provider.dart (Home feed)

Both providers need to filter videos before emitting to UI.
